# ⚙️ Configurações do Sistema

Este projeto apresenta uma **interface de configurações**, permitindo que o usuário selecione opções como **Wi-fi, Bluetooth e Dados Móveis**. Cada opção possui um **toggle estilizado** com CSS puro.

## 📝 Recursos
- Toggles animados para cada opção.  
- Textos alinhados ao lado de cada checkbox.  
- Design moderno usando gradiente de fundo e flexbox.  

## 🛠 Tecnologias
- HTML5  
- CSS3  

## 🎯 Objetivo
Criar um painel de configurações com **checkboxes estilizados** para estudo de layout e interação visual.
